# aSkelHCoRe2
aSkelHCoRe,is  a framework of Reconfigurable Algorithmic Skeletons for Co-design in Heterogeneous Systems using CPU, GPU and FPGA, which leverages Cole’s skeleton paradigm to provide high-level abstractions for structured and transparent parallelism. Implemented in OpenCL/Python, it offers reusable Pipeline and Master–Slave skeletons.
